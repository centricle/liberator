# Privacy Policy

Liberator does not collect or transmit any user data.

## Data Storage

The extension stores your checkbox preferences locally in Chrome's storage. This data never leaves your device.

## Network Activity

This extension makes no network requests. All functionality runs entirely on your local machine.

## Permissions Explained

- **activeTab**: Used only to inject fixes into the current page when you click the button
- **storage**: Used only to remember your checkbox preferences locally
- **scripting**: Used only to run the fix scripts on the current page
